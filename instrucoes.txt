Para iniciar o projeto com node
-> npm init -y

Para instalar o fastify (porque fastify ao invés de express? mais rápido e mais bem mantido)
-> npm i fastify

Para instalar o typescript
-> npm i typescript

Para instalar o types node
-> npm i @types/node -D
IPORTANTE: Pode instalar os dois juntos com 
-> npm i typescript @types/node -D

Para instalar o pino
-> npm i pino-pretty

Subir um docker
-> docker compose up

Deixar o docker rodando direto
-> docker compose up -d

Ver as imagens do docker que estão rodando
-> docker ps

Parar o docker se deletar as bases
-> docker compose stop

Para instalar o drizzle (ORM)
-> npm i drizzle-kit -D

Para instalar o drizzle (ORM - Versão para produção) + drive do postgres
-> npm i drizzle-orm pg

Para instalar o zod e a integração do zod com fastify
-> npm i zod fastify-type-provider-zod

Para instalar o swagger
-> npm i @fastify/swagger

Para instalar a interface do swagger
-> npm i @fastify/swagger-ui

Para instalar o scalar afim de melhorara a interface da documentação
-> npm install @scalar/fastify-api-reference

Para instalar faker JS
-> npm i @faker-js/faker -D

Para instalar o vitest
-> npm i vitest -D

Para instalar o supertest
-> npm i supertest -D

Para instalar o supertest para rodar no ts (para tudo que roda no ts mas não foi feito em ts, tem que instalar @types/ + biblioteca)
-> npm i @types/supertest -D

Para instalar o dotenv-cli (ler o .env pelo vitest)
-> npm i dotenv-cli -D

Para instalar o coverage v8
-> npm i -D @vitest/coverage-v8
